PART 2 — PLAN LIMITATIONS (CRITICAL)
==================================================

Add a new table: plans
Fields:
- id
- name
- price
- max_agents
- max_logs_per_day
- max_commands_per_day
- allow_network_isolation (boolean)
- allow_process_kill (boolean)
- allow_file_scan (boolean)
- allow_endpoint_download (boolean)

Add a new table: usage_tracking
Fields:
- user_id
- date
- agents_registered
- logs_sent
- commands_executed

Enforce plan limits:
- When agent registers → check max_agents
- When logs are sent → check max_logs_per_day
- When commands are executed → check max_commands_per_day
- When user tries to isolate network → check allow_network_isolation
- When user tries to kill processes → check allow_process_kill
- When user tries to scan files → check allow_file_scan
- When user tries to download agent installer → check allow_endpoint_download

If limit exceeded:
Return error: “Your current plan does not allow this action. Please upgrade.”
