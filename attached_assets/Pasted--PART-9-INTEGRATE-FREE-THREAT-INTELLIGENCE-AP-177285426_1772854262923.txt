PART 9 — INTEGRATE FREE THREAT INTELLIGENCE APIs
==================================================

Integrate the following free, commercially-allowed threat intelligence APIs to enhance AegisAI360 detection capabilities:

1. AbuseIPDB (IP Reputation)
----------------------------
API: https://api.abuseipdb.com/api/v2/check
Use cases:
- Check if an IP is malicious
- Show abuse confidence score
- Show categories (SSH brute force, DDoS, spam, etc.)

Backend:
- Create service: services/threatIntel/abuseIpdb.js
- Create endpoint: POST /api/threat-intel/ip
- Input: ip_address
- Output: reputation score, categories, last reported date

Plan enforcement:
- Only available if plan.allow_threat_intel = true

2. AlienVault OTX (Threat Indicators)
-------------------------------------
API: https://otx.alienvault.com/api/v1/indicators
Use cases:
- Lookup IP, domain, URL, or file hash
- Retrieve threat pulses
- Retrieve malware families

Backend:
- Create service: services/threatIntel/otx.js
- Create endpoint: POST /api/threat-intel/otx-lookup
- Input: indicator (ip, domain, url, hash)
- Output: threat pulses, malware info, reputation

Plan enforcement:
- Only available if plan.allow_threat_intel = true

3. URLScan.io (URL Analysis)
----------------------------
API: https://urlscan.io/api/v1/scan/
Use cases:
- Analyze suspicious URLs
- Detect phishing, malware, redirects
- Screenshot and DOM analysis (optional)

Backend:
- Create service: services/threatIntel/urlscan.js
- Create endpoint: POST /api/threat-intel/urlscan
- Input: url
- Output: scan results, verdict, risk score

Plan enforcement:
- Only available if plan.allow_threat_intel = true

4. Google Safe Browsing (Malicious URL Detection)
-------------------------------------------------
API: https://safebrowsing.googleapis.com/v4/threatMatches:find
Use cases:
- Detect phishing URLs
- Detect malware URLs
- Detect unwanted software

Backend:
- Create service: services/threatIntel/googleSafeBrowsing.js
- Create endpoint: POST /api/threat-intel/safebrowsing
- Input: url
- Output: threat type, platform, severity

Plan enforcement:
- Only available if plan.allow_threat_intel = true

5. MalwareBazaar (File Hash Lookup)
-----------------------------------
API: https://mb-api.abuse.ch/api/v1/
Use cases:
- Check SHA256 hash reputation
- Identify malware families
- Retrieve YARA rule matches

Backend:
- Create service: services/threatIntel/malwareBazaar.js
- Create endpoint: POST /api/threat-intel/hash
- Input: sha256 hash
- Output: malware family, signature, tags

Plan enforcement:
- Only available if plan.allow_threat_intel = true
